# SIVEN 策略模块

大道量化系统 - SIRENUSDT 多空闭环策略

## 策略核心

- **相似K线形态匹配**: 寻找历史重复规律
- **缠论顶底拐点**: 精准识别转折时机
- **多空闭环**: 开多→平多→开空→平空

## 策略特点

1. 同一时间只持一个方向
2. 不双向持仓
3. 等待明确的平仓信号后再反向开仓

## 信号定义

- **开多信号**: 底分型 + 相似历史上涨概率 > 60%
- **平多信号**: 顶分型 + 相似历史下跌概率 > 60%
- **开空信号**: 顶分型 + 相似历史下跌概率 > 60%
- **平空信号**: 底分型 + 相似历史上涨概率 > 60%
