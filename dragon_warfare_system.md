## 三、缠论分析体系

### 3.1 结构层级

```
笔 (Fenbi)
  ↓
线段 (Xianduan)
  ↓
中枢 (Zhongshu)
  ↓
趋势 (Trend)
```

### 3.2 买卖点系统

| 买点 | 条件 | 级别 |
|------|------|------|
| 一买 | 底背驰后的最低点 | 激进 |
| 二买 | 次低点 | 稳健 |
| 三买 | 突破中枢后回踩不破 | 中枢上沿 |
| **卖点** | **条件** | **级别** |
| 一卖 | 顶背驰后的最高点 | 激进 |
| 二卖 | 次高点 | 稳健 |
| 三卖 | 跌破中枢后回抽不破 | 中枢下沿 |

### 3.3 背驰识别

背驰判断标准：
- **价格**：创新高/新低
- **力度**：MACD红/绿柱面积减小
- **量能**：成交量萎缩
- **时间**：笔的K线数量减少

## 四、量化开仓标准

### 4.1 强制要求

| 项目 | 最低要求 |
|------|----------|
| **盈亏比** | ≥ 1:4 |
| **止损位** | 结构安全位（不易被洗） |
| **止盈位** | 按规律和盈亏比执行 |

### 4.2 开仓条件

**多头开仓（做多）：**

```
✓ 均线多头排列（MA5 > MA10 > MA30）
✓ MACD金叉（MACD > Signal）
✓ RSI < 70（未超买）
✓ 出现缠论买点（一买/二买/三买）
✓ 盈亏比 ≥ 1:4
```

**空头开仓（做空）：**

```
✓ 均线空头排列（MA5 < MA10 < MA30）
✓ MACD死叉（MACD < Signal）
✓ RSI > 30（未超卖）
✓ 出现缠论卖点（一卖/二卖/三卖）
✓ 盈亏比 ≥ 1:4
```

### 4.3 止损策略

| 止损类型 | 位置 | 说明 |
|----------|------|------|
| 结构止损 | 最近支撑/压力位外 | 缠论结构保护 |
| 固定止损 | 入场价 - 4% | 极端情况 |
| 跟踪止损 | 移动止盈 | 保护利润 |

### 4.4 止盈策略

| 止盈位 | 目标 | 条件 |
|--------|------|------|
| TP1 | 1:1 | 盈亏比1:1处 |
| TP2 | 1:2 | 盈亏比1:2处 |
| TP3 | 1:4 | 盈亏比1:4处（目标位） |

## 五、龙头因子体系

### 5.1 个性因子档案

每个品种建立专属档案，记录：

```json
{
  "historical_extremes": {
    "all_time_high": "历史最高价",
    "all_time_low": "历史最低价",
    "max_rise_ratio": "最大涨幅",
    "max_fall_ratio": "最大跌幅"
  },
  "price_patterns": {
    "common_rises": ["常见上涨形态"],
    "common_falls": ["常见下跌形态"],
    "breakout_patterns": ["突破形态"]
  },
  "volume_characteristics": {
    "volume_spike_threshold": 2.0,
    "accumulation_indicator": "主力吸筹特征"
  },
  "support_resistance": {
    "key_support_levels": ["关键支撑"],
    "key_resistance_levels": ["关键阻力"]
  }
}
```

### 5.2 共性规律提炼

从个性因子中提炼共性：

1. **启动特征**
   - 连续放量突破均线
   - MACD零轴附近金叉
   - 成交量放大至均量2倍以上

2. **加速特征**
   - 沿5日均线上涨
   - MACD红柱持续放大
   - 量价齐升

3. **见顶特征**
   - 放量滞涨
   - 顶背驰（价格新高，MACD不新高）
   - 跌破5日均线
