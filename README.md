# 大道量化策略系统

> 大道至简,顺势而为

## 项目简介

大道量化策略系统是一个基于 BTC 1小时K线的量化交易策略系统,遵循大道心法:

```
无动无静,无生无灭,无去无来,无是非,无住无往。
直观,屏息注念,一念不生,湛于末时。
了了分明,一尘不染。
```

## 核心原则

- **应无所住而生其心**
- 不预测,只识别;不预判,只跟随
- 心不住于涨跌,不住于多空,不住于盈亏,不住于境相
- 以K线为相,以量价为证,以结构为凭,以历史为镜
