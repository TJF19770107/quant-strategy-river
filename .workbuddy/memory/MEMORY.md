# 工作记忆

## 大道量化策略系统 V3.0（万能模板）

### 专属品种数据库
- 配置文件: `symbols_db.json`
- 支持品种: RIVERUSDT、BANANAS31USDT、POWERUSDT、PIPPINUSDT（可随时新增）
- 添加品种: `add_symbol_to_db("标的名称")`

### 模板功能
- 通用分析模板: `dao_strategy.py` (V3.0)
- 使用方式: `python dao_strategy.py [标的名称]`
- 支持: 币安合约API (fapi) + 现货API双通道
- 数据: 优先读取本地 data/[标的]_1h.csv，自动补齐最新K线
- 重试机制: 3次自动重试 + 断点续传
- 输出: data/[标的]_1h.csv, reports/[标的]_1h_chan.html, reports/[标的]_backtest_report.md
- 核心规则：只做顺势多单，盈亏比必须≥1:4

### 唤醒口令
- "分析 [标的名称]" 或 "大道量化" → 自动执行完整分析流程
- "加入品种 [标的名称]" → 将新标的添加到品种数据库

## 大道龙头币品种库与万能战法体系

### 品种库
- 配置文件: `dragon_tokens_db.json`
- 龙头币: RIVERUSDT、BANANAS31USDT、POWERUSDT、PIPPINUSDT、BEATUSDT、AIAUSDT、MYXUSDT、COAIUSDT
- 个性因子档案: `dragon_tokens/[币种]_profile.json`
- 缠论分析: `dragon_tokens/[币种]_analysis.json`

### 缠论分析模块
- 文件: `chan_theory.py`
- 功能: 笔/线段/中枢/背驰/买卖点识别

### 万能龙头战法
- 文件: `dragon_warfare_system.md`
- 开仓标准: 均线多头 + MACD金叉 + RSI<70 + 缠论买点 + 盈亏比≥1:4
- 止损策略: 结构止损（支撑/压力位外）、固定止损（-4%）、跟踪止损
- 止盈策略: TP1(1:1)、TP2(1:2)、TP3(1:4目标位)
