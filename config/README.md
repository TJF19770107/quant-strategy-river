# 配置文件目录

## 目录结构

```
config/
├── README.md           # 本文件
├── database.yaml      # 数据库配置
├── params.yaml        # 策略参数配置
└── api_keys.yaml      # API密钥配置 (不提交到Git)
```

## 配置说明

### 数据库配置 (database.yaml)

用于配置数据存储相关参数。

### 策略参数配置 (params.yaml)

用于配置策略的核心参数。

### API密钥配置 (api_keys.yaml)

用于存储交易所API密钥,**不会提交到Git**。
