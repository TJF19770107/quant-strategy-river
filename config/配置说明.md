## 策略参数配置

### RIVER 策略默认参数 (BTCUSDT 1h)

```json
{
  "RIVER_WINDOW": 20,
  "PROB_THRESHOLD": 0.55,
  "VOL_THRESHOLD": 1.2,
  "STOP_LOSS_PCT": 0.02,
  "TAKE_PROFIT_PCT": 0.08,
  "MAX_POSITION": 1.0,
  "USE_TRAILING_STOP": false,
  "TRAILING_TRIGGER": 0.05,
  "MAX_CONSECUTIVE_LOSSES": 999,
  "MIN_TRADES_PER_MONTH": 5
}
```

### 参数说明

| 参数 | 默认值 | 说明 |
|------|--------|------|
| RIVER_WINDOW | 20 | RIVER 高低点窗口周期 |
| PROB_THRESHOLD | 0.55 | ML 预测概率阈值 |
| VOL_THRESHOLD | 1.2 | 成交量放大倍数 |
| STOP_LOSS_PCT | 0.02 | 止损百分比 (2%) |
| TAKE_PROFIT_PCT | 0.08 | 止盈百分比 (8%) |
| MAX_POSITION | 1.0 | 最大仓位比例 |
| USE_TRAILING_STOP | false | 是否使用移动止盈 |
| TRAILING_TRIGGER | 0.05 | 移动止盈触发点 |
| MAX_CONSECUTIVE_LOSSES | 999 | 连续亏损限制 |
| MIN_TRADES_PER_MONTH | 5 | 最小交易频率 |
